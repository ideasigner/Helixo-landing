Helixo — 本地 HTML 生产力工具

TL;DR — AI 时代，HTML 正在成为本地生产力文档的通用格式。但传统浏览器对本地 HTML 的支持几乎为零：没有文件导航、没有多标签、没有实时同步。Helixo 是一款专为本地 HTML 工作流设计的桌面工具，让 HTML 从"网页载体"变成"可管理、可协作、可扩展的生产力文档"。
原始需求 基于完整的产品功能文档，将 Helixo 的 6 大模块、11 项子功能、技术定位和目标用户整理为可视化的功能说明页面。

Helixo 主界面总览 — 文件夹即工作区

01文件夹即工作区

打开一个文件夹，Helixo 将其视为完整的工作区。文件发现、导航、收藏一气呵成。

自动扫描与索引

打开一个文件夹，Helixo 自动识别其中所有 HTML 文件，按目录结构呈现。支持小文件过滤、MHTML 识别，告别在 Finder 里逐个双击打开的时代。左侧文件树 + 搜索框，导航效率提升一个量级。

收藏夹 + 最近文件

常用文件一键收藏，最近打开的文档随时回溯。左侧边栏在"文件树 / 收藏夹 / 最近"三种视图间自由切换，高频文件永远在手边。

02像 IDE 一样管理文档

将代码编辑器的高效操作模式引入文档管理。多标签、分屏、工作区恢复，全部到位。

多标签 + 拖拽分屏

任意拖拽标签页即可左右分屏，同时对比两份文档，或一边写、一边看效果。双编辑器分组、标签页 drag-and-drop，操作丝滑程度不输现代代码编辑器。

工作区自动恢复

关闭即保存，打开即继续。Helixo 自动记忆你的工作状态：打开的文件、标签页顺序、分屏布局、侧边栏宽度、缩放比例，下次启动一键回到上次离开的地方。

03实时同步创作流

AI Agent 自动生成内容时，Helixo 是连接"生成"与"查看"的桥梁。文件变更即时同步，创作循环无需中断。

文件变更自动刷新

开启自动刷新后，外部任何修改（包括 AI Agent 自动生成或修改内容）都会即时反映在预览中。不再需要手动切回浏览器按 F5，这是 AI 协作工作流的关键基础设施。

预览和源码一键切换，修改 HTML 后直接保存。写代码、看效果、再调代码，在一个窗口内完成循环，无需在编辑器和浏览器之间来回跳转。

04深度浏览与调试

Helixo 不只是查看器。大纲导航、元素检查——为开发者提供浏览器不愿给本地文件的深度工具。

智能文档大纲

自动从 HTML 页面中提取 h1-h6 标题结构，生成可点击的目录导航。长文档阅读时，左侧大纲随滚动实时高亮当前位置，点击即可跳转——浏览器永远不会为本地 HTML 提供这种体验。

元素检查器

点击预览中的任意元素，实时查看 Tag、ID、Class、Color、Background、Font-Size 等样式信息。前端开发者调试样式时，无需再开浏览器 DevTools。

05输出与协作

从代码到设计稿，从文档到素材。Helixo 打通本地 HTML 与外部协作工具之间的最后一公里。

灵活导出任意宽度

支持导出完整页面为 PDF、PNG/JPEG（可自定义视口宽度）。做分享、归档、演示素材时，不再受浏览器打印样式的限制。

06为专注而设计

工具不应争夺注意力。Helixo 在需要时隐身，在需要时顺手。

一键专注模式

整屏只展示文件本身内容，工具栏、侧边栏、标签栏全部隐去。适合演示、截图、沉浸式阅读，也适合向他人展示时聚焦内容本身。

Electron · Zero Backend · Local-First
