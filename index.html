Before / After · 范式转变

从浏览到工作

Before · 旧模式

Finder → 双击 → 浏览器

  • 在 Finder 里逐个找到 HTML 文件
  • 双击后用浏览器打开，标签页迅速膨胀
  • 修改代码后手动切回浏览器按 F5
  • 无法同时对比两份文档
  • 导出时受限于浏览器打印样式
After · 新模式

Helixo 工作区

  • 打开文件夹，自动索引所有 HTML
  • 多标签 + 拖拽分屏，IDE 级管理
  • 文件变更自动刷新，零延迟同步
  • 左右分屏同时对比多份文档
  • 灵活导出 PDF / PNG / SVG

Phase 01 · 文件管理

文件夹即工作区

打开一个文件夹，Helixo 自动识别其中所有 HTML 文件，按目录结构呈现。

支持小文件过滤、MHTML 识别。常用文件一键收藏，最近文档随时回溯。左侧边栏在"文件树 / 收藏夹 / 最近"三种视图间自由切换。

"像 IDE 一样管理文档——多标签、拖拽分屏、工作区自动恢复。"
— Helixo 设计哲学
