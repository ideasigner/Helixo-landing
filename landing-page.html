Core Features

一切为 HTML 工作流而设计

从文件浏览到导出交付，覆盖完整的 HTML 设计闭环

📁

本地文件浏览器

递归扫描文件夹内的 HTML 文件，空目录自动修剪。三栏布局：文件树、预览区、工具栏。

👁

实时预览 & 编辑

iframe 实时渲染，一键切换源码编辑。修改后自动刷新，支持 25%–200% 缩放预览。

🖼

一键导出 Figma

将网页捕获为 Figma 可粘贴的 HTML 数据，转为可编辑的设计图层。支持本地文件和 URL。

📑

标签页 & 分屏

多标签同时打开，拖拽排序。左右两组分屏对比，空组自动折叠节省空间。

📤

多格式导出

Figma、PDF、PNG/JPEG、SVG、打印。支持视口截图和整页截图，可自定义宽度。

🌐

URL 模式

直接输入网址打开网页，支持 React/Vue SPA。逐段滚动触发懒加载，确保截图完整。

Use Cases

谁在用 Helixo？

三种典型工作流，覆盖设计与开发全场景

01

静态页面设计师

"我做了很多 HTML 落地页，每次都要开 Chrome 预览然后截图发给客户。Helixo 可以直接在分屏里对比两个版本，还能一键导出高清 PNG。"

02

Figma 设计稿还原

"客户给了我一个网页，我需要把它还原成 Figma 设计稿。Helixo 的 Figma 导出可以直接把整个网页粘贴进 Figma，变成可编辑的图层。"

03

本地 HTML 开发

"我在写一个纯 HTML/CSS 的文档站点，Helixo 的文件树自动扫描所有 HTML 文件，点一下就能预览，改代码自动刷新，比开服务器方便多了。"
